===== Supermarket System =====

    - ตรวจสอบจำนวนสินค้าในสต๊อก
3. Sales/Checkout System
    - เลือกสินค้า / จำนวนที่ลูกค้าซื้อ(สกาย)
    - คำนวนราคา(สกาย)
    - ลดราคา/โปรโมชั่น(ทางเลือก)(ไบโอ)
    - ใบเสร็จ(ไบโอ)
4. Sales Report(ออม)
   - รายงานยอดขายหลังจากออกโปรแกรม
